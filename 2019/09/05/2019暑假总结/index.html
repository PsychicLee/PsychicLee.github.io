2019暑假总结

暑期培训总结

快乐的暑假趁我一不注意就过完了，说长不长，说短不短，自我感觉这个暑假过得还算充实，虽然不是每天都忙忙碌碌晚睡早起，但也算是没有荒废掉太多时间。

1. 养成了良好的作息习惯

高中的作息习惯是晚上十一点半睡觉，早上六点左右起床，中午半个小时午休，现在看来真的是极其规律。但是自从上了大学，睡觉时间逐渐从十一点变成十一点半、十二点，最后经常都是不到两点不放下手机睡觉。晚睡导致的就是晚起，起床时间一再推迟，以至于就算没早课都经常是踩点进教室的。

但是这个暑假，白天长时间盯着电脑屏幕，晚上还会去跑跑步，加上早上八点就得起床，如此一来，晚上回到宿舍洗完澡之后已经是很累很困了，自然而然也就早早上床睡觉。

传说21天就能养成一个习惯，一个月下来，我已经习惯了十二点前睡觉，八点前起床。

亲身体验之后，我坚信同样是睡七个小时，早睡早起比晚睡晚起来得舒服，起码起床后不会感觉浑身酸痛。

2. 跟工作室其他成员搞好了关系

刚加入工作室，先不说跟师兄师姐，就算是同届的另外五个人（甚至还有两个人跟我同班），我都感觉很生疏，坐在一起都会有一点尴尬的感觉。我不知道该如何与他们相处，感觉每个人好像都很高冷。

但是，一个暑假下来，我深刻认识到，其实大家就是一群逗比，只不过先前都在隐藏本性而已。

良好合作的前提是互相了解、互相信任、互相帮助，既然是一个团队，就应该一起沙雕。

3. 学到了更多的知识

这个是肯定的。

不过除此之外，我感受到了记笔记的好处。暂且不说记的笔记之后还会不会重温，起码我过一段时间再看看放笔记的文件夹，能直观地感受到我这段时间都干了些什么。

其实我觉得笔记更大的用处在于，刚学完的第二天极容易遗忘知识点，但是重新去翻看详细之极的文档又有点浪费时间，此时如果有前一天记的笔记，一边浏览笔记，一边回忆起之前学习的东西，效率会高很多。

4. 去珠海溜达了一圈

大爷说想看海，于是我们去了珠海。

海虽然是不怎么看得清楚（毕竟天黑了），但是沙滩是走过了，海风是吹过了。也就是在那一天，我才突然明白了广州比我家那边热的原因————广州没有海风。

我是个看柯南都能看到不敢上厕所的人，当天晚上居然跟着看了一部恐怖片，不可思议啊。

5. 拿了驾照

虽然我一直自称秋名山第一车手，但是我一直到这个暑假才拿到了驾照。

我以后也是能跟人家吹我是四科都一次性过的了。

6. 各自有梦想，各自做决定

高中有几个同学一直都有保持联系，都说高中的友谊是最纯真的，现在看来，那几个家伙确实是值得深交的。

有两个家伙，选择了去当兵。起码两年见不到他们了呀。

虽说大学生当兵待遇挺好，但是谁也不知道我们的未来会是怎么样的。希望十年之后，我们都不会后悔现在做出的决定。
